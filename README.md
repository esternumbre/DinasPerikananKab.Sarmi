# DinasPerikananKab.Sarmi
Website profil dan layanan publik Dinas Perikanan Kabupaten Sarmi.

## Demo
https://esternumbre.github.io/DinasPerikananKab.Sarmi/

## Fitur Utama
- Tampilan homepage modern dan responsif (desktop & mobile).
- Top bar informasi dengan tanggal dan waktu real-time.
- Navigasi lengkap dengan dropdown profil dan mega menu layanan.
- Hero slider interaktif dengan CTA utama.
- Section dinamis: layanan, berita, tentang, galeri, dan CTA.
- Branding dan konten dapat diubah melalui `assets/data/profil.json` (logo, nama instansi, teks, link, dsb).
- Footer informatif dengan menu navigasi, informasi publik, dan layanan cepat.

## Teknologi yang Digunakan
- HTML5
- Tailwind CSS (CDN)
- CSS kustom (`css/style.css`)
- JavaScript vanilla (`assets/js/main.js`)
- JSON sebagai sumber data konten (`assets/data/profil.json`)
- Alpine.js (CDN)
- Font Awesome
- Google Fonts (Inter)
